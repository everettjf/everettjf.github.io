RepoRead

macOS

Read GitHub repositories like a book — no clone, no setup.

RepoRead is a small native macOS app for reading code. Paste a GitHub URL, get a tree of files with proper syntax highlighting, and keep reading offline afterwards. Built for the times you just want to look at a project without setting one up.

Download on the Mac App Store Download DMG Join Discord Feedback

Or install with Homebrew.

Features

Paste a URL

Drop in a GitHub repo URL and start reading. No clone, no fork, no checkout.

Monaco-powered viewer

Real syntax highlighting for 50+ languages, the same engine VS Code uses.

File tree + icons

A familiar tree on the left, type-aware icons, and quick navigation.

Offline cache

Repos cache locally so you can keep reading on a plane.

Native macOS app

Small, fast, opens instantly. Not an Electron wrapper.

Homebrew install

One command to install and one command to update.
