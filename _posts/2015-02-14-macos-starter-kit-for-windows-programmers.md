### macbook pro
In early October 2014, I bought a MacBook Pro and the book "MacTalk: Life and Meta-Programming". From then on, a C++ programmer who had been hauling bricks under Windows for 5 years accidentally stepped into the world of OSX.

### A Few Bits of Common Sense
- Power button (the button at the very top-right of the keyboard)
- Finder (corresponds to Windows Explorer — that's the name of the file manager)
- Command (corresponds to the Windows key on the Windows system)
- Option (corresponds to the Alt key on Windows)
- Trackpad gestures (click the Apple in the top-left corner, System Preferences, Trackpad — there are animated demos here)
- img files (Mac installers, similar to Windows portable apps, similar to Windows iso files)
- Fullscreen and maximize (there's no maximize, only fullscreen. Or rather, fullscreen and maximize are similar)
- Command-Tab to switch between different programs
- Control-Tab to switch within the same program

### macbook pro
2014年10月初，买了macbook pro和《MacTalk人生元编程》，从此一位在Windows下搬了5年砖的C++程序猿不小心跨入了OSX的世界。

### 几个小常识
- 开机按钮（键盘最右上角的按钮）
- Finder（对应Windows的资源管理器，就是资源管理器的名字）
- Command（对应Windows系统的Windows键）
- Option（对应Windows的Alt键）
- 触摸板手势（点击左上角的苹果，系统偏好设置，触摸板，这里有动画演示）
- img文件（mac的安装程序，类似Windows的绿色软件，类似Windows的iso文件）
- 全屏与最大化（没有最大化，只有全屏。或者说全屏与最大化类似）
- Command-Tab 不同程序间切换
- Control—Tab同程序间切换
