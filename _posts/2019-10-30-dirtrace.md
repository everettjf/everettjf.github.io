# Background

There's often a need to analyze the size that files take up within a folder (for example, analyzing a mobile app's installation package size). For a long time, [GrandPerspective](http://grandperspectiv.sourceforge.net/) basically met this need, but while using it I often wished I could see the parent-child relationship of folders. [FolderSizeView](https://everettjf.github.io/2019/07/07/foldersizeview/) solved this need.

However, FolderSizeView's performance is too poor. When the folder has too much content (for example analyzing the file sizes inside `Xcode.app`), it can't handle it. Actually, after FolderSizeView was released, people kept reporting that it couldn't produce results when there was too much file content. After discussing, I found that people were actually comparing FolderSizeView against some disk analysis tools. Since the original design intent was only to analyze app installation packages — and it solved my personal need — I didn't want to invest time optimizing it.

# Installation

dirtrace is developed in Python, and you can install it with pip.

```
pip install dirtrace
```

If your Python environment doesn't have pip installed, just download
https://bootstrap.pypa.io/get-pip.py
this file, then `python get-pip.py` to finish the installation.

# Usage

Just two steps (So easy — never have to worry about not knowing how to analyze a folder again).

## Step 1: Generate the json

```
dirtrace -d <directory-path>
dirtrace -d <directory-path> -o <output-json-path>
```

For example:
```
dirtrace -d /Application/Contacts
or
dirtrace -d /Application/Contacts -o ~/Desktop/trace.json
```

Note: the output format is json. If you don't specify the `-o` parameter, it will generate `trace.json` in the current working directory.

## Step 2: View

Open the Chrome browser (it must be Chrome), and open
```
chrome://tracing/
```

As shown:

![-w270](/media/15723983993848.jpg){:width="540" height="72"}

`Drag` the `trace.json` file you just generated `into Chrome`.

Done! Try operations like `w a s d f`.
